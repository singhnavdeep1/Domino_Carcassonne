README


Pour pouvoir exécuter notre jeu, directement dans le dossier AyadiJana-SinghNavdeep effectuer la commande suivante: 


-pour le jeu avec l’interface graphique:
              javac Launcher.java
et ensuite
        java Launcher




-pour le domino terminal
        javac view/DominoTextuel.java
et ensuite
        java view/DominoTextuel
